Invoke the [angular-component-development](../../skills/angular-component-development/) skill.

## Run Command

```bash
npm start
```

Server runs at: **http://localhost:4200**

## What It Does

1. Starts Angular development server
2. Enables hot module reloading (HMR)
3. Watches for file changes
4. Auto-refreshes browser on save
5. Compiles TypeScript in memory
6. Reports compilation errors in browser and console

## Development Features

- **Fast refresh** — Changes visible in < 1 second
- **Source maps** — Debug TypeScript, not compiled JavaScript
- **Console errors** — Displayed in browser dev tools
- **CSS reloading** — Styles update without full refresh
- **Angular DevTools** — Available in Chrome DevTools

## Stopping the Server

Press `Ctrl+C` in the terminal to stop.

## Debugging

- Open Chrome DevTools: `F12`
- Set breakpoints in TypeScript
- Use Angular DevTools extension for component inspection
- Check browser console for runtime errors
